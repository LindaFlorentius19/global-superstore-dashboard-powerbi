# Global-superstore-dashboard-powerbi
Interactive Power BI dashboard analyzing sales, profit, discount, quantity, and shipping costs across categories, segments, and ship modes.

## Overview
This Power BI dashboard provides insights into sales performance, profit trends, customer segments, categories, and shipping modes using the Global Superstore dataset.

## Key Metrics
- Total Sales: 4.94M
- Total Profit: 556.3K
- Total Quantity: 70K
- Total Discount: 2.88K
- Shipping Cost: 527.76K

## Dashboard Features
- Sales by Segment
- Sales by Category
- Sales by Ship Mode
- Monthly Sales Trend
- Monthly Profit Trend
- Top Performing Sub-Categories
- Interactive Category Filter

## Tools Used
- Power BI
- DAX
- Power Query
- Data Modeling
